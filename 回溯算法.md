# backtracking回溯算法
## 适用题型：列出所有可能的选择如：组合，切割，部分子集问题，排列，棋盘

e.g. leetcode 17.电话号码组合

算法模板：
```
void backtracking(List<Sting> arg1, Integer arg2...){
         if(满足出口的终止条件){
               收集结果；
               rerun;
  }
       for(集合元素){
               处理节点；
              调用递归函数backtracking；
              状态重置操作；
}
```
